### 4. 주요 기능
- 개인회원 기능
	- 회원가입, 로그인
	- 마이페이지(작성, 조회, 수정)
- **실천하기**
	- **게시글 조회, 페이징 기능**
	- **게시글 작성, 수정, 삭제 및 댓글 작성**
	- **좋아요 기능**
	- **지도 보기, 이메일 발송**
- 위어스학교
	- 달력 보기
	- 게시글 작성, 조회, 검색
	- 좋아요 기능
- 쇼핑하기
	- 물품 목록 보기 및 가격순 정렬
	- 문의 보기 및 문의 글 작성하기
	- 장바구니
	- 주문 및 결제
- 고객센터
	- 게시글 조회

- 관리자 기능
	- 회원 수, 게시글 수 등 현황보기
	- 회원보기 및 회원정보 추가/수정
